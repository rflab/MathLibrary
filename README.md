<h2>概要</h2>
3Dグラフィックスの計算に必要な、ベクトル、クオータニオンの計算を行うライブラリです。
自作ゲーム向けなので、速度重視でそこそこ正しく動けばいいやというスタンスで作成されています。

<h2>使い方</h2>
ライブラリを利用するにはmath/index.hをインクルードしてください。
